## Controls
`W/S` - move forward/backward<br/>
`D/A` - move right/left<br/>
`V/C` - move up/down<br/>
`arrows` - rotation<br/>
`[` - next shader<br/>
`]` - previous shader<br/>
`R` - reload shader<br/>
`T` - reset time<br/>
`F` - freeze time<br/>
`U` - run shader debugger for pixel under cursor<br/>
`space` - pause rendering<br/>
`escape` - exit<br/>
`left mouse button` - rotate by mouse<br/>
